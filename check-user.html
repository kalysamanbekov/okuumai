Проверка информации о пользователе

Введите email пользователя для получения информации о дате регистрации и статусе триала.

Информация о пользователе будет отображена здесь.

Важно:

Для получения полной информации о пользователе требуются административные права в Firebase. В реальном приложении эту функциональность следует реализовать через защищенные Cloud Functions.
